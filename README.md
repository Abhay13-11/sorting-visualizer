# Sorting Algorithm Visualizer

An interactive web application to visualize how different sorting algorithms work using animated bars and real-time comparisons.

Live Demo
https://abhay13-11.github.io/sorting-visualizer

---

## Features

* Interactive sorting visualization
* Bubble Sort, Merge Sort, and Quick Sort
* Adjustable array size
* Speed control for animations
* Gradient animated bars
* Swap sound feedback
* Sorting time measurement
* Algorithm comparison mode
* Time complexity graph using Chart.js
* Modern glassmorphism UI

---

## Algorithms Implemented

### Bubble Sort

* Time Complexity: O(n²)
* Space Complexity: O(1)

### Merge Sort

* Time Complexity: O(n log n)
* Space Complexity: O(n)

### Quick Sort

* Average Time Complexity: O(n log n)
* Worst Case: O(n²)
